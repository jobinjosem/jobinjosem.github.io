Customize your powershell using oh-my-posh

  2021-12-04   wsl   wsl kubernetes terminal Word Count: 425 words Read Time: 2 minutes

In a previous post, I explained how to customize the bash prompt in WSL using kube-ps1. Turns out that kube-ps1 is a bit outdated and there are other cool tools available in the marketplace like oh-my-posh which is widely used among the community. In this post, I will explain how I personalized my PowerShell prompt using oh-my-posh.

Below are the steps I followed to personalize my Powershell prompt.

  1. Install oh-my-posh.
Install-Module oh-my-posh -Scope CurrentUser -AllowPrerelease
  1. Install a font that supports Powerline symbols.

Download the Caskaydia Cove Nerd Font. Extract the zip file, select them all and right-click to install.

  1. Set the Caskaydia font on the settings of the windows terminal.

Navigate to Windows Terminal –> Settings –> Powershell –> Appearance –> Font face. Then choose CaskaydiaCove Nerd Font and click save.

  1. Preview and set the theme.

We can preview all themes by running the Get-PoshThemes command and applying a theme of your choice using the Set-PoshPrompt <theme name> command. The theme will be applied on running the Set-PoshPrompt command but won’t persist beyond the current session.

  1. Override the theme settings

I had some requirements like showing my current Azure subscription name, the Kubernetes cluster I am connected to and the current time in the PowerShell prompt. So I decided to export the theme and tweak it as per my requirement. We can output the current theme to a json file which can be used to tweak and store as your custom theme.

Export-PoshTheme -FilePath "c:\theme\mytheme.omp.json" -Format json

We can copy segments from here and modify the color, symbols, and style. The configuration I used is available here

  1. Apply the custom theme.

To apply the settings permanently, we can add the theme configuration to your Powershell profile.

Open PowerShell profile using code $PROFILE and add the below lines.

Import-Module oh-my-posh
Set-PoshPrompt -Theme "c:\theme\mytheme.omp.json"

The terminal must look like this if you are using the config file that I used.

  1. Add kubectx + kubens

This will help to switch between different kubernetes clusters and namespaces, We need to download the kubectx, kubens and fuzzyfinder utilities and add the executables to the $PATH. Then set alias by adding below lines in $PROFILE

Set-Alias -Name x -Value '<PATH>\kubectx.exe'
Set-Alias -Name n -Value '<PATH>\kubens.exe'

With kubectx we can easily switch between different Kubernetes clusters by typing x and press enter. A list of available Kubernetes cluster contexts in your kubecfg will be shown.

Similarly pressing n will list all the namespaces in the selected cluster and you can select from the list.
